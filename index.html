My Projects

Project 1: SimplePasswordValidator

A simple tool designed to check if a password meets basic security standards. It ensures that a password is at least 8 characters long, contains at least one uppercase letter, one lowercase letter, and one number. The program works by asking the user to input a password, then running it through a function that checks these requirements. If the password meets all the criteria, the program confirms it as valid. Otherwise, it explains why the password is invalid, listing the specific rules that need to be followed. It's a straightforward way to encourage stronger, more secure passwords.

Project 2: Link 'n' Give

A mobile application that connects givers and receivers to facilitate community-driven donations and resource sharing. Givers can upload photos and details of items they wish to donate, while receivers can use a location-based radar to find and access nearby donations easily. The app enables direct communication between users, making the process seamless and efficient. By promoting the reuse of items, Link’n’Give helps reduce waste, supports sustainability, and provides essential resources to those in need. This platform fosters a more connected and compassionate community while encouraging environmental responsibility.

Project 3: MHoare Partition

This project enhances the IntroSort algorithm by integrating a Modified Hoare Partitioning Method to tackle the code space efficiency challenges highlighted by David Musser. The proposed modification optimizes memory usage while preserving high sorting performance.
